## 🎯 3 STEPS TO START

### 1️⃣ Add Your OpenAI API Key (REQUIRED)

Open `backend\.env` and add your key:

```env
OPENAI_API_KEY=sk-your-actual-openai-key-here
```

**Get your key**: https://platform.openai.com/api-keys

### 2️⃣ Open the Application

Click or open in browser:
```
http://localhost:3000
```

### 3️⃣ Analyze Your First Repository

1. Enter a GitHub URL (try: `https://github.com/fastapi/fastapi`)
2. Click "Analyze"
3. Wait 30-60 seconds
4. View generated documentation!
